## 我们的使命 | Our Mission

**开放知识计划（Open Knowledge Project）** 致力于编写和传播 **免费、开放、共享** 的教学文档，让知识的获取不再受限于经济门槛，让人类积累的知识财富得以跨世代传承。

我们相信：
- **知识应当自由流动** — 每一个人都有权利免费获取优质的学习资源
- **分享创造价值** — 协作编写让文档更准确、更完整、更易理解
- **开源精神永续** — 开放的不仅是大门，更是通向未来的道路

> *We believe that knowledge belongs to everyone.*

## 为什么选择我们？| Why Choose Us?

- **完全免费** — 所有文档免费阅读、下载，无任何付费门槛
- **开源协作** — 所有项目采用开放许可证，欢迎提交 Issue 与 Pull Request
- **中文优先** — 专注中文技术文档，降低学习曲线的第一步门槛
- **持续更新** — 随着技术演进和社区反馈，文档持续迭代优化

## 如何参与 | How to Contribute

我们欢迎任何形式的贡献！

1. **Star / Fork** — 在 GitHub 上关注我们，将仓库 Fork 到你的账户
2. **Issue** — 发现文档中的错误、遗漏，或有新主题推荐？[提交 Issue](https://github.com/open-knowledge-project/.github/issues)
3. **Pull Request** — 直接编辑文档内容，通过 PR 参与协作编写
4. **传播分享** — 如果你觉得这些资源有价值，将它们分享给需要的朋友和同学

> *Every contribution, no matter how small, helps break down the barrier to knowledge.*
